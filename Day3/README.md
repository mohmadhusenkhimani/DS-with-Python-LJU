# 🏠 Homework Programs

## H1. 3-Digit Lock Combination

### Problem Statement

Generate every possible 3-digit lock combination using the digits **1, 2, and 3** without repeating any digit.

### Concepts Covered

- Backtracking
- Recursion
- Permutations

---

## H2. Folder Size Calculator

### Problem Statement

Calculate the total size of a folder by recursively visiting all files and subfolders.

### Concepts Covered

- Recursion
- File Handling
- `os` Module
- Directory Traversal

---

## H3. Climbing Stairs

### Problem Statement

Find the total number of distinct ways to reach the Nth stair when you can climb either **1 step** or **2 steps** at a time.

### Concepts Covered

- Recursion
- Dynamic Programming Concept
- Mathematical Problem Solving
